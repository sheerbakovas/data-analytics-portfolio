# Data Analytics Portfolio

**Автор:** Ирина Щербакова

Практические проекты по анализу данных с упором на автоматизацию отчётности, работу с API и построение интерактивных дашбордов.

# Проекты

## 01 — E-commerce Hourly Email Report  

Автоматизированный почасовой отчёт по e-commerce данным.

**Что реализовано:**
- Очистка и подготовка clickstream-данных
- Расчёт ключевых e-commerce метрик
- Почасовая агрегация
- Автоматическая отправка email-отчёта
- Хранение состояния обработанных периодов

**Ключевые метрики:**
- Показы
- Клики
- Конверсии
- Выручка
- Средний чек
- Воронка: просмотр → корзина → покупка

---

## 02 — Adzuna Telegram Reporting Bot  

Telegram-бот для ежедневного мониторинга рынка вакансий через Adzuna API.

**Что реализовано:**
- Интеграция с внешним API
- Обработка и агрегация данных
- Расчёт средней зарплаты
- Анализ топ-локаций и компаний
- Автоматическая ежедневная отправка отчёта
- Генерация Excel-файла с детализацией

**Ключевые метрики:**
- Общее количество вакансий
- Средняя зарплата
- Доля вакансий с указанной зарплатой
- Топ-локации
- Топ-компании

---

## 03 — Sales Conversion Optimization Dashboard (DataLens)  

Интерактивный BI-дашборд для анализа эффективности рекламных кампаний.

**Что реализовано:**
- Полная воронка продаж
- Анализ рекламных кампаний
- Расчёт CTR, CR, CPA, Approval rate
- Анализ аудитории по возрасту и полу
- Heatmap (пол × возраст)
- Анализ эффективности по интересам
- Детализация по объявлениям
- Система фильтров

## Подход к работе

Во всех проектах реализован полный аналитический цикл:

1. Подготовка и очистка данных  
2. Расчёт бизнес-метрик  
3. Автоматизация или визуализация  
4. Интерпретация результатов
